⚙️ Setup Admin Panel SEKALI SAJA

Masukkan konfigurasi GitHub untuk mengaktifkan Admin Panel. Data ini disimpan di browser Anda (localStorage) dan tidak dibagikan ke siapapun.

📋 Cara mendapatkan GitHub Token:
1. Login ke github.com → klik foto profil → Settings
2. Scroll ke bawah → Developer settings
3. Personal access tokens → Tokens (classic)
4. Generate new token (classic)
5. Nama: kavling-admin, centang: repo → klik Generate
6. Salin token yang muncul (hanya tampil sekali!)
